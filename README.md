# gulp-browserify-template
##Первый запуск
```bash
npm install -g gulp ios-sim
npm install # установка зависимостей
make init
```

## Запуск сервера для разработки
```bash
gulp # запускает сервер с поддержкой BrowserSync на 3000 порту
```
